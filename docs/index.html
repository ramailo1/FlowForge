Installation Guide

Follow these simple steps to get started with FlowForge:

  1. Step 1: Clone the Repository

    Open your terminal and run:

    git clone https://github.com/ramailo1/FlowForge.git

  2. Step 2: Navigate to the Directory

    Switch to the cloned folder:

    cd FlowForge

  3. Step 3: Load as Unpacked Extension

    Steps for Chromium-based browsers:

    • Open your browser and go to the extensions page.
    • Chrome: chrome://extensions
    • Edge: edge://extensions
    • Enable Developer mode (top right).
    • Click Load unpacked.
    • Select the FlowForge directory.
    • You're ready to go!
