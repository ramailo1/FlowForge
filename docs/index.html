Installation Guide

Get started with FlowForge in just a few steps.

01

Clone the Repository

Start by cloning the FlowForge repository to your local machine.

git clone https://github.com/ramailo1/FlowForge.git
02

Navigate to Directory

Move into the FlowForge folder to access the project files.

cd FlowForge
03

Load as Unpacked Extension

Install FlowForge in your Chromium-based browser.

  • Open chrome://extensions or edge://extensions.
  • Enable Developer mode.
  • Click Load unpacked and select the FlowForge folder.
  • Enjoy FlowForge!
