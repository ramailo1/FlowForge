Installation Guide

Follow these simple steps to get the FlowForge extension up and running in your browser.

  1. Step 1: Clone the Repository

    Open your terminal or command prompt and execute the following command to clone the FlowForge repository to your local machine:

    git clone https://github.com/ramailo1/FlowForge.git

  2. Step 2: Navigate to the Directory

    Change your current directory to the newly cloned FlowForge repository:

    cd FlowForge

  3. Step 3: Load as Unpacked Extension

    This extension is designed to be loaded as an unpacked extension in Chromium-based browsers (e.g., Google Chrome, Microsoft Edge, Brave). Here's how:

    • Open your browser and navigate to the extensions page.
    • For Chrome, type chrome://extensions in the address bar.
    • For Edge, type edge://extensions in the address bar.
    • Enable Developer mode (usually a toggle switch in the top right corner).
    • Click on the Load unpacked button.
    • In the file dialog, select the FlowForge directory that you cloned in Step 1.
    • The FlowForge extension should now appear in your list of extensions and be ready for use!
