About FlowForge

Whether you're researching a topic, managing your tools, or navigating your favorite websites in a routine way, FlowForge observes your repeated actions and offers to automate them—no coding or setup required.

FlowForge passively detects patterns in your navigation across websites and suggests them as reusable “Flows”. It provides AI-powered suggestions, multi-step automations, and ensures privacy by keeping all data local on your device. You can also create custom Flows using its manual mode.

How to Install from GitHub (for Developers)

If you prefer to install FlowForge directly from the GitHub repository, follow these steps:

  1. Clone the Repository: Open your terminal or command prompt and run the following command to clone the FlowForge repository: 
    git clone https://github.com/ramailo1/FlowForge.git
  2. Navigate to the Directory: Change into the cloned directory: 
    cd FlowForge
  3. Load as Unpacked Extension in Edge:
    • Open Microsoft Edge.
    • Go to edge://extensions.
    • Turn on Developer mode (usually a toggle in the top right corner).
    • Click on Load unpacked.
    • Browse to the FlowForge directory you cloned (f:\FlowForge) and select it.
    • The extension should now be loaded and visible in your extensions list.
